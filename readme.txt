== Installation ==
1. Create your free Survey-2-Sale account via https://survey2sale.com/signup.html
2. Upload the entire s2s folder to the /wp-content/plugins/ directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Access the Survey-2-Sale menu item from \"Settings > Survey-2-Sale\"
5. Click \"Enable Survey-2-Sale\"
